# JavaScript-Course

A collection of projects and reflections from my journey into full-stack development. This repository showcases the transition from theoretical concepts to functional, interactive applications.

## 🚀 Core Competencies
* **Structure:** Semantic HTML5 for accessible and SEO-friendly content.
* **Styling:** Responsive CSS3 designs and layouts.
* **Logic:** Dynamic interactivity and DOM manipulation using JavaScript (ES6+).

## 🛠 Featured Projects
* **Portfolio Website:** A personal hub designed to showcase my creative and technical work.
* **Interactive Tic-Tac-Toe:** A logic-based game implementing win-condition algorithms.
* **Calculator App:** A functional tool focused on arithmetic logic and clean UI.
* **Pizza Menu:** A stylized web page utilizing CSS positioning and layout techniques.

## 🧠 Key Learnings
This course focused on bridging the gap between syntax and solution. By building applications from the ground up, I gained significant experience in:
- **Problem Solving:** Navigating complex logic and debugging code.
- **Modern Workflow:** Preparing for advanced frameworks like React and Angular.
- **Creative Execution:** Translating design ideas into working digital products.
